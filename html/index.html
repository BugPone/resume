Simple bot used by 200 chatrooms (2019)

Node.JS SQL + AWS Linux

What's was this? I built and formerly hosted a humorous chatroom utility bot with AWS / GCP servers and MySQL databases using the Discord.Js API.
What did it do? It lets users flag one another, where if enough unique users flag one user, it performs an action on the flagged user. I built Socrate's tradjedy.
 - MySQL to support settings editable by the chatroom owner and log usage statistics to gain potential insights. I did not host this version publicly.

Notable challenges...
- Hosting and Maintence: Trying to keep uptime and functionality despite deprecated APIs and mandatory changes.
- Managing cloud costs: Reading cost reports and trimming unused resources.
- Memory leak: Keeping track of the votes that are occuring and ensuring they are deleted when they votes expire or are done. Undeleted votes would have piled up in RAM, eating memory until the next reboot.
- Protecting users: Not leaking keys (probably) and preventing odd use cases from manipulating vote counts.
